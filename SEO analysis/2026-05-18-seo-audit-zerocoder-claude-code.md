## 3. Структура заголовков (H1–H6)

| Тег | Текст |
|-----|-------|
| H1 | БАЗОВЫЙ |
| H1 | БИЗНЕС |
| H1 | ВИП |
| H2 | ПРОВОДИМ ИССЛЕДОВАНИЯ ПО ИИ СОВМЕСТНО С ЛУЧШИМИ ВУЗАМИ СТРАНЫ |
| H2 | НАШИ ПРЕМИИ И РЕЙТИНГИ |
| H2 | Нужна консультация по обучению? |

**Критическая ошибка:** 3 тега `<H1>` на одной странице.  
Поисковики ожидают один H1, который описывает главную тему документа. Текущие H1 — названия тарифов (БАЗОВЫЙ, БИЗНЕС, ВИП), что искажает семантику.

**Рекомендация:** оставить один H1 с целевым ключом («Практический курс по вайб-кодингу на Claude Code» или «Курс по Claude Code и вайб-кодингу»), а названия тарифов перевести в `<H3>` или `<H2>`.

**Критическая ошибка:** ни одно изображение не содержит атрибута `alt`.  
Поисковый бот не понимает контент картинок, а скринридеры не могут озвучить их пользователям.

**Рекомендация:** добавить описательные alt-тексты хотя бы для ключевых изображений (герой-блок, иллюстрации курса, логотипы).
