## 7. Технические метрики (Core Web Vitals)

| Метрика | Значение | Статус |
|---------|----------|--------|
| TTFB | 547 мс | ⚠️ Выше нормы |
| FCP | **7 536 мс** | ❌ Критически медленно |
| DOMContentLoaded | 1 229 мс | ✅ Норма |
| Load Complete | 5 256 мс | ❌ Медленно |
| Transfer Size | 103 КБ | ✅ Норма |
| JS Size | 507 КБ | ❌ Много |
| CSS Size | 561 КБ | ❌ Много |
| Image Size | 244 КБ | ✅ Норма |
| Total Resources | 121 | ⚠️ Много запросов |

**Критическая проблема:** FCP составляет 7.5 секунд. Пользователи видят белый экран почти 8 секунд.  
Вероятные причины (Next.js):
- Большой объём JS/CSS (507 + 561 = ~1 МБ)
- 121 сетевой запрос
- Тяжёлые шрифты (9 файлов .woff2)
- Возможно, блокирующий рендеринг код

**Рекомендации по скорости:**
1. Включить `next/font` с `display: swap` для шрифтов
2. Разделить CSS на критический (inline) и некритический (lazy-load)
3. Code-splitting для JS-чанков
4. Оптимизировать порядок загрузки ресурсов
5. Проверить работу сервера (TTFB 547 мс — медленный ответ)
