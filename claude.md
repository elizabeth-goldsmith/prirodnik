# CLAUDE.md — Правила работы для Лизы

## Обращение
Обращайся к Лизе по имени — Лиза.

## Роль
Лиза — менеджер проектов. Задачи связаны с планированием, разработкой и запуском проектов.

## Обязательные правила

### План перед действием
- Перед любой задачей показывай план в виде шагов.
- Жди подтверждение Лизы. Без подтверждения — не действуй.

### Ограничения (без разрешения Лизы — нельзя)
- Не удаляй файлы.
- Не отправляй сообщения во внешние системы.
- Не выходи за пределы рабочей папки `ai-lessons`.

### Формат ответов
- Отвечай коротко.
- Результат работы — таблица с колонками: **Задача | Результат | Время**.

## Инструменты и среда
- Таблицы — Excel
- Код — VS Code
- Просмотр и редактирование файлов - Zed

## Правила нейминга файлов
Рабочая формула имени файла:
`[дата — опционально] — [тема] — [тип].md`

Примеры:
- `2026-04-21-sales-weekly-report.md`
- `client-onboarding-checklist.md`

## Git
Не используется.

## Язык
Все ответы и файлы — на русском языке.
